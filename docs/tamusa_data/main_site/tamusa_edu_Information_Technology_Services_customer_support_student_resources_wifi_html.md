## How To Connect 
The following information includes steps for Student to connect your Windows or Apple devices to the campus WiFi. To connect your Android/Chromebook to the WiFi, scroll down. 
###  ** Configure The TAMU-SA-Student WIFI Connection: **
  1. Navigate to your device’s WiFi settings 
  2. Select the TAMU-SA-Student WiFi and click Connect. 
  3. In the username box, type in Jaguar Username 
  4. In the password box, type in Jaguar Password 
  5. It will ask to "Continue connecting?" click Connect.   
** *Apple devices will need to trust a certificate after connecting*  **


###  **Connecting a Chromebook/Android to the TAMU-SA-Student WiFi**
  1. Navigate to your device’s WiFi settings 
  2. Select the TAMU-SA-Student WiFi and click Advanced 
  3. Type in Jaguar credentials for the Identity: Jaguar username Password: Jaguar password 
  4. Select PEAP as your EAP Method 
  5. Select Do Not Check/Don’t Vaildate under the Server CA Certificate 
  6. Select MSCHAPV2 as the Phase 2 authentication 
  7. Domain: tamusa.edu (if required) 
  8. Optional: Save Identity and Password 
  9. Click on Connect


**Troubleshooting Tips:**
  1. If error message says, "Can't connect" right click on the WIFI name TAMU-SA-Student and click Forget. Try connecting to the WiFi again. 
  2. Restart device
  3. If the above troubleshooting fails to get you connected then you may need to run updates for your device to make sure that it is fully updated.
